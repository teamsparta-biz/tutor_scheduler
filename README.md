## 개요

강사 일정 관리 서비스. 운영팀이 교육 과정에 강사를 배정하고, 같은 날 중복 배정을 자동으로 차단한다. 강사 본인도 자기 일정을 확인할 수 있다.

**대상 사용자:**
- 운영팀(관리자): 전체 강사/교육 관리, 배정 수행
- 강사: 자기 일정 확인

**핵심 기능 (MVP):**
- 강사 등록/프로필 관리
- 교육 일정 등록 (하루/며칠 단위)
- 강사 배정 + 같은 날 중복 차단
- 캘린더 뷰 (월간/주간)

**업무 규칙:**
- 강사 1명 = 하루에 1개 교육만 가능
- 교육 1개에 여러 강사 배정 가능 (반별: A반, B반 등)
- 교육은 하루짜리 / 며칠짜리 모두 존재

**기술 스택:**
- 프론트엔드: React (Vite)
- 백엔드: Python + FastAPI
- DB/인증: Supabase (PostgreSQL + Google OAuth)
- 배포: Railway (앱 코드) + Supabase (DB/인증)

**규모:** 강사 50명 이상
