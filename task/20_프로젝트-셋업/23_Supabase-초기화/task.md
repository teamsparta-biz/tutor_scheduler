# Supabase 초기화

> Supabase 프로젝트를 생성하고 DB 테이블을 마이그레이션한다

## 목표

- Supabase 프로젝트가 생성됨
- 설계된 DB 스키마대로 테이블이 생성됨
- Google OAuth 제공자가 활성화됨

## 상세

- [ ] Supabase 프로젝트 생성 (supabase.com)
- [ ] 13_DB-스키마-설계의 결과를 SQL로 작성
- [ ] Supabase SQL Editor에서 테이블 생성 실행
- [ ] RLS(Row Level Security) 정책 기본 설정
- [ ] Google OAuth 제공자 활성화 (Supabase 대시보드 → Authentication → Providers)
- [ ] API Key, URL 등 환경변수 값 확인 및 기록

## 참고

- Google Cloud Console에서 OAuth 클라이언트 ID/Secret 발급 필요
- Supabase 무료 티어 제한: 500MB DB, 50K 월간 활성 사용자
