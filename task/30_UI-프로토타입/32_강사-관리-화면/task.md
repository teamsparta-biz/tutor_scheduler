# 강사 관리 화면

> 강사 목록, 검색/필터, 등록/수정 모달을 React 컴포넌트로 구현

## 목표

- 강사 목록 테이블 + 검색/필터 + 등록/수정 모달 완성

## 상세

- [x] 강사 목록 페이지: 테이블(이름, 이메일, 연락처, 전문분야, 상태), 검색, 활성/비활성 필터
- [x] 강사 등록/수정 모달: 이름, 이메일, 전화번호, 전문분야, 활성 상태 입력
- [x] 빈 검색 결과 상태 처리
- [x] 강사별 일정 캘린더 모달: 이름 왼쪽 아이콘 버튼 → 월간 가용성+배정 캘린더
