# UI 프로토타입

> React 컴포넌트로 전체 화면 프로토타입을 제작하고, A/B안으로 핵심 화면의 레이아웃을 비교한다

## 목표

- 모든 주요 화면의 React 프로토타입이 완성됨
- 핵심 3개 화면(대시보드, 배정, 캘린더)에 A/B 토글 제공
- 하드코딩 mock 데이터로 UI 확인 가능

## 상세

- [x] 31_로그인-대시보드: 로그인 페이지 + 대시보드 A/B안
- [x] 32_강사-관리-화면: 강사 목록, 검색/필터, 등록/수정 모달
- [x] 33_교육-일정-화면: 교육 목록, 상태 필터, 교육 상세/날짜 패널
- [x] 34_강사-배정-화면: 매트릭스/카드 A/B안, 중복 경고
- [x] 35_캘린더-뷰-화면: 풀그리드/분할뷰 A/B안, 월/주 전환

## 참고

- React + Tailwind CSS로 직접 구현 (코드 재사용 가능)
- mock 데이터: 강사 5명, 교육 3개, 배정 8건
- `npm run dev` → `http://localhost:5173` 에서 확인
