# UI 프로토타입

> HTML/CSS만으로 전체 화면 목업을 제작하고, UI를 확인한 뒤 프론트엔드 개발에 착수한다

## 목표

- 모든 주요 화면의 HTML/CSS 목업이 완성됨
- 사용자가 화면 구성과 흐름을 확인하고 승인함
- 프론트엔드 개발 시 참고할 UI 가이드가 확보됨

## 상세

- [ ] 31_로그인-대시보드: 로그인 페이지 + 메인 대시보드
- [ ] 32_강사-관리-화면: 강사 목록, 등록/수정 폼
- [ ] 33_교육-일정-화면: 교육 목록, 등록/수정 폼
- [ ] 34_강사-배정-화면: 교육별 강사 배정 UI, 중복 경고
- [ ] 35_캘린더-뷰-화면: 월간/주간 캘린더 레이아웃

## 참고

- 기능 로직 없이 순수 HTML/CSS로 구현
- 브라우저에서 직접 열어서 확인
- 확인 완료 후 React 컴포넌트로 전환
