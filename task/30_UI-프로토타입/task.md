# UI 프로토타입

> React 컴포넌트로 전체 화면 프로토타입을 제작. A/B 비교 후 B안 확정, 추가 기능 반영 완료

## 목표

- 모든 주요 화면의 React 프로토타입이 완성됨
- A/B 비교 후 B안 확정 (대시보드, 배정, 캘린더)
- 하드코딩 mock 데이터로 UI 확인 가능

## 상세

- [x] 31_로그인-대시보드: 로그인 페이지 + 대시보드 B안 (오늘/이번 주 탭)
- [x] 32_강사-관리-화면: 강사 목록, 검색/필터, 등록/수정 모달, 강사별 일정 캘린더
- [x] 33_교육-일정-화면: 교육 목록, 상태 필터, 교육 상세/날짜 패널
- [x] 34_강사-배정-화면: B안 카드 리스트, 중복 경고
- [x] 35_캘린더-뷰-화면: B안 분할뷰, PM 표시, 미배정 경고
- [x] 36_강사-가용성-화면: 가용성 등록 캘린더 + 관리자 현황 목록

## 참고

- React + Tailwind CSS로 직접 구현 (코드 재사용 가능)
- mock 데이터: 강사 5명, 교육 3개, 배정 8건, 가용성 32건
- `npm run dev` → `http://localhost:5173` 에서 확인
- 정책 결정: 가용성 페이지는 향후 권한별 완전 분리 (강사=등록만, 관리자=현황만)
