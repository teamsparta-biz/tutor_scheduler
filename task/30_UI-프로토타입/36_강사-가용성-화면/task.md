# 강사 가용성 화면

> 강사 가용성 등록 캘린더 + 관리자 현황 목록을 React 컴포넌트로 구현

## 목표

- 강사가 날짜별 가용 상태(가능/불가/미정)를 등록하는 캘린더
- 관리자가 특정 날짜의 전체 강사 가용 현황을 한눈에 파악하는 목록

## 상세

- [x] 일정 등록 탭: 강사 선택 → 월간 캘린더, 날짜 클릭으로 상태 순환 (가능→불가→미정)
- [x] 배정된 날짜 보호: 이미 배정된 날짜는 "배정됨" 표시, 클릭 불가
- [x] 가용성 현황 탭: 날짜 선택 → 전체 강사 목록 (상태 + 배정 교육)
- [x] 배정 시 자동 상태 덮어쓰기: 배정된 강사는 "배정됨"으로 표시 (가용 상태 무시)
- [x] 색상 구분: 녹색(가능), 빨강(불가), 주황(미정), 파랑(배정됨)

## 참고

- 파일: `pages/Availability.tsx`
- mock 데이터: 가용성 32건 (4 강사 × 8 날짜)
- 정책 결정: 향후 권한별 완전 분리 (강사=등록만, 관리자=현황만)
- 라우트: `/availability`, 사이드바 "강사 가용성"
