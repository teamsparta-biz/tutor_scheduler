# 로그인 & 대시보드 화면

> 로그인 페이지와 메인 대시보드의 HTML/CSS 목업을 제작한다

## 목표

- Google 로그인 버튼이 포함된 로그인 페이지 목업 완성
- 로그인 후 보이는 메인 대시보드 레이아웃 완성
- 관리자/강사 각각의 대시보드 차이 표현

## 상세

- [ ] 로그인 페이지: Google 로그인 버튼, 서비스 로고/설명
- [ ] 관리자 대시보드: 사이드바 메뉴(강사 관리, 교육 관리, 배정, 캘린더), 요약 통계
- [ ] 강사 대시보드: 내 일정 요약, 이번 주/이번 달 배정 현황
- [ ] 공통 헤더: 사용자 정보, 로그아웃 버튼
