# 교육 프론트엔드

> React 교육 관리 화면을 구현한다

## 목표

- 교육 목록 페이지에서 상태 필터(진행중/예정/완료)가 동작함
- 교육 등록/수정 폼에서 날짜 범위와 반 설정이 가능함

## 상세

- [ ] 교육 목록 페이지 (30_UI-프로토타입 기반 → React 전환)
- [ ] 교육 등록 폼: 과정명, 날짜 선택(단일/범위), 반 추가/삭제, 장소, 메모
- [ ] 교육 수정 폼
- [ ] 교육 삭제 (배정 존재 시 경고)
- [ ] 교육 상세 페이지: 일자별 배정 현황 표시
- [ ] API 연동
