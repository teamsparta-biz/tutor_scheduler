# 강사 프론트엔드

> React 강사 관리 화면을 구현한다

## 목표

- 강사 목록 페이지에서 검색/필터/페이지네이션이 동작함
- 강사 등록/수정 폼이 동작하고 API와 연동됨
- 삭제 시 확인 다이얼로그 표시

## 상세

- [ ] 강사 목록 페이지 (30_UI-프로토타입 기반 → React 컴포넌트 전환)
- [ ] 강사 등록 폼 (모달 또는 별도 페이지)
- [ ] 강사 수정 폼
- [ ] 강사 삭제 (확인 다이얼로그 → API 호출)
- [ ] API 연동 (fetch 또는 axios)
- [ ] 로딩/에러 상태 처리
