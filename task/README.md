# task/

프로젝트 작업 항목을 관리합니다.

## 참조

| 주제 | 문서 |
|------|------|
| 상위 폴더 | [../README.md](../README.md) |

## 구조

```
task/
├── _template/          → 템플릿 폴더
│   └── task.md
├── 10_태스크명/        → 메인 태스크
│   ├── task.md
│   └── data/           → 관련 자료 (필요시)
├── 11_하위태스크/      → 10번의 하위 태스크
│   └── task.md
└── 20_다음태스크/
    └── task.md
```

## 번호 체계

| 번호 | 용도 |
|------|------|
| 10, 20, 30... | 메인 태스크 |
| 11, 12, 13... | 10번의 하위 태스크 |
| 21, 22, 23... | 20번의 하위 태스크 |

## 상태 정의

| 상태 | 설명 |
|------|------|
| 대기 | 아직 시작하지 않음 |
| 진행중 | 작업 중 |
| 완료 | 작업 완료 |
| 보류 | 일시 중단 |
