# 배정 프론트엔드

> React 강사 배정 화면을 구현한다

## 목표

- 교육별 날짜/반 매트릭스에서 강사를 배정/해제할 수 있음
- 이미 배정된(불가용) 강사가 시각적으로 구분됨
- 중복 배정 시도 시 에러 메시지 표시

## 상세

- [ ] 배정 화면 (30_UI-프로토타입 기반 → React 전환)
- [ ] 교육 선택 → 날짜별/반별 배정 매트릭스 표시
- [ ] 강사 선택 드롭다운: 가용 강사만 표시 또는 불가용 강사 비활성화
- [ ] 배정 추가: 강사 선택 → API 호출 → 매트릭스 업데이트
- [ ] 배정 해제: 클릭 → 확인 → API 호출
- [ ] 중복 에러 처리: API 400 응답 시 사유 표시
